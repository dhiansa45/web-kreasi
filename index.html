Tentang Kami

Kenapa belajar coding penting?

Keterampilan utama di era digital yang membuka peluang karier, meningkatkan kemampuan berpikir logis, mendorong inovasi, dan membantu seseorang beradaptasi dengan perkembangan teknologi.

Melalui coding, kita tidak hanya menjadi pengguna teknologi. Akan tetapi juga pencipta solusi yang dapat memberikan dampak positif bagi berbagai aspek kehidupan.

Metode Belajar Coding

List materi belajar coding sesuai dengan jenjang pendidikan.

- Begginer -

a. Scratch (MIT) : Dalam platform scratch peserta didik dapat di arahkan dalam menggunakan blok visual untuk membuat cerita, animasi maupun game yang menarik. b. Code.org : Terdapat berbagai permainan interaktif seperti Minecraft atau Frozen yang dapat membantu sistem pembelajaran logika pemrograman. c. Thynker : Terdapat berbagai jenis permainan dan proyek yang menarik guna meningkatkan skill dalam belajar coding. d. Blockly :Berupa editor pemrograman visual yang dikembangkan oleh Google dimana memungkinkan pengguna dalam membuat program sederhana tanpa mengetik kode

- Intermediate -

1. Belajar Bahasa Pemrograman Dasar: a. Pilih bahasa seperti Python (mudah dipahami pemula). b. Ajarkan dasar-dasar seperti variabel, tipe data, loop, dan kondisi. 2. Gunakan Platform Edukasi: a. Replit: Platform online untuk coding langsung. b. Kaggle (Intro to Python): Untuk eksplorasi data sederhana. 3. Proyek Sederhana: a. Buat kalkulator sederhana. b. Program untuk menampilkan pola bintang. c. Game berbasis teks seperti "Tebak Kata". 4. Kenalkan Konsep Pemecahan Masalah: a. Ajarkan bagaimana memecahkan masalah dengan algoritma sederhana. b. Latih debugging untuk mengatasi error. 5. Mulai Kompetisi: a. Partisipasi dalam kompetisi coding seperti Google Code-In (jika tersedia).

- Advanced -

1. Pelajari Bahasa Pemrograman Lanjutan: a. Web Development: HTML, CSS, JavaScript. b. Mobile Development: Kotlin (Android) atau Swift (iOS). c. Data Science: Python dengan library seperti Pandas dan NumPy. 2. Gunakan Sumber Belajar Online: a. FreeCodeCamp: Untuk proyek berbasis web. b. Coursera/edX: Kursus lebih mendalam. c. Github: Untuk kolaborasi dan manajemen kode. 3. Proyek Menengah: a. Buat aplikasi sederhana, seperti aplikasi daftar tugas (to-do list). b. Buat website portofolio pribadi. c. Mulai eksplorasi dengan API atau data scraping. 4. Ikut Program atau Bootcamp: a. Ikut serta dalam bootcamp coding. b. Bergabung dengan komunitas developer lokal. 5. Mempersiapkan Karier: a. Pelajari pengembangan portofolio proyek. b. Ikuti magang atau proyek freelance kecil.

Manfaat Program Belajar Coding

Dengan diadakannya program pembelajaran coding berbasis web learning IoT Applications. Hal ini sangat bermanfaat jika diberlakukan dari mulai jenjang SD sampai ke jenjang selanjutnya. Adapun manfaat yang dapat diperoleh diantara lain adalah :

- Manfaat Program Belajar Coding -

1. Membangun Literasi Terkait dengan Pentingnya Teknologi Sejak Dini


2. Meningkatkan Keterampilan Dalam Berpikir Kritis Siswa Disaat Memecahkan Masalah


3. Memperkenalkan Konsep Belajar IoT


4. Dapat Mendorong Kreativitas dan Inovasi Peserta Didik


5. Dapat Turut Aktif Dalam Menumbuhkan Sikap Kemandirian dan Kerjasama Tim


6. Persiapan Terhadap Karier di Masa Depan


7. Akses Belajar Menjadi Lebih Mudah dan Fleksibel


8. Mendorong Minat dan Bakat Siswa di Bidang STEM (Science, Technology, Engineering and Math)


9. Dapat Membantu Para Guru Dalam Meningkatkan Keterampilan Belajar Siswa


10. Membangun Generasi Penerus Bangsa yang Melek Akan Digitalisasi dan Siap Untuk Menghadapi Tantangan di Masa Depan
